Справка мобильного приложения "Магазин гитар"

Регистрация нового пользователя:

Описание

Экран регистрации позволяет создать нового пользователя, указав имя и сумму на балансе. Основные элементы интерфейса:

  • Поле имени пользователя: Введите уникальное имя для пользователя.
  • Поле суммы: Введите сумму, которая будет добавлена на баланс.
  • Кнопка "Зарегистрироваться": Сохраните данные и перейдите к экрану входа.

Как зарегистрироваться?

  1. Введите имя пользователя в первое текстовое поле.
  2. Введите сумму в поле "Сумма". Убедитесь, что сумма введена в корректном формате (например, 1234.56).
  3. Нажмите кнопку "Зарегистрироваться".
  4. Если все данные введены правильно, вы перейдете на экран входа.
  5. В случае ошибки (например, пустые поля или некорректный формат суммы) появится уведомление с соответствующей информацией.

Технические особенности

  • Сохранение данных: Используется UserPreferences для сохранения имени пользователя и суммы на балансе.
  • Обработка ошибок: Некорректный ввод данных (например, пустое поле или неверный формат суммы) вызывает уведомления с описанием ошибки.

Главный экран:

Описание

Главный экран приложения "Music Shop Guitar" отображает баланс пользователя и список категорий товаров, доступных для просмотра и покупки. Основные элементы интерфейса включают:

  • Баланс: Отображение текущего баланса пользователя.
  • Список категорий: Перечень категорий товаров (например, "Гитары", "Аксессуары к гитаре").

Основной функционал

На главном экране доступны следующие функции:

  1. Просмотр баланса: Отображается баланс пользователя в формате "Ваш баланс составляет: X.XX₽". Баланс берется из сохраненных настроек приложения.
  2. Выбор категории: Нажатие на категорию открывает экран со списком продуктов, принадлежащих к этой категории.

Категории товаров

Каждая категория содержит список продуктов. Например:

  • Гитары: Электрогитары, акустические гитары, бас-гитары, классические гитары.
  • Аксессуары к гитаре: Тюнеры, медиаторы, каподастры, ремни, струны и другое.

Как использовать экран?

Для работы с главным экраном:

  1. Просмотрите ваш текущий баланс в верхней части экрана.
  2. Выберите категорию, нажимая на нужный элемент списка.
  3. После выбора категории откроется новый экран с перечнем продуктов в этой категории.

Дополнительная информация

Если вы хотите вернуться на этот экран из других частей приложения, просто нажмите кнопку "Назад" на вашем устройстве.
